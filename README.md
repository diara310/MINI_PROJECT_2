<h2> Pilihan Pertama ADMIN </h2>

<img width="1200" height="472" alt="pilihan pertama" src="https://github.com/user-attachments/assets/34bd1613-95c6-429f-b213-dacaf3e26317" />
Pada tampilan pertama, pengguna diminta untuk memasukkan username dan password.
<br>
	•	Jika login berhasil sebagai Admin, maka sistem menampilkan menu utama.
  <br>
	•	Admin bisa mengelola data CD/DVD (tambah, lihat, update, hapus) dan juga melihat data peminjaman.

<h2> Pilihan Kedua </h2>

<img width="1199" height="244" alt="pilihan kedua" src="https://github.com/user-attachments/assets/4bcfb1cd-e19d-452a-ba96-695a8856c8d6" />
Admin memilih menu 1 (Tambah CD/DVD).
<br>
	•	Sistem meminta input berupa ID CD/DVD, Judul, dan Jumlah Stok.
  <br>
	•	Jika data berhasil disimpan, muncul pesan konfirmasi: “Data CD/DVD berhasil ditambahkan”.

<h2> Pilihan Ketiga </h2>

<img width="1206" height="319" alt="pilihan kdetiga" src="https://github.com/user-attachments/assets/b638b6ce-c331-4cee-b399-ab11fc21e90b" />
Admin memilih menu 2 (Lihat Data CD/DVD).
<br>
	•	Sistem akan menampilkan daftar CD/DVD yang sudah disimpan.
  <br>
	•	Contoh output: ID: 12, Judul: CD, Stok: 12.

<h2> Pilihan Keempat </h2>

<img width="1201" height="287" alt="pilihan keempat" src="https://github.com/user-attachments/assets/aa89a9bf-0b89-4965-8a06-c4f3a1274d32" />
Admin memilih menu 3 (Update Data CD/DVD).
<br>
	•	Sistem meminta ID CD/DVD yang ingin diupdate.
  <br>
	•	Admin bisa mengganti judul baru dan jumlah stok baru.
  <br>
	•	Setelah diubah, sistem menampilkan pesan: “Data CD/DVD berhasil diupdate”.

<br>
	•	Sistem meminta ID CD/DVD yang ingin dihapus.
  <br>
	•	Jika data ditemukan, maka data akan dihapus dan sistem menampilkan pesan: “Data CD/DVD berhasil dihapus”.
